# ModelLedger Roadmap

## Vision
AI compliance should be a one-command byproduct of normal MLOps workflows — not a separate audit exercise.

## ✅ Shipped
- Model lineage tracking (training data → code → model → evaluation)
- EU AI Act compliance report generation
- NIST AI RMF compliance mapping
- Fully offline — no database, no cloud
- 48+ tests

## 🔨 In Progress
- [ ] MLflow integration for automatic lineage capture
- [ ] HTML report output format
- [ ] ISO 42001 (AI Management System) compliance mapping

## 📋 Planned — Q2 2025
- [ ] Weights & Biases experiment pull integration
- [ ] GitHub Actions workflow for auto-report on model release
- [ ] Risk classification wizard (EU AI Act Articles 6–10)
- [ ] PDF report improvements with executive summary

## 📋 Planned — Q3 2025
- [ ] SageMaker experiment integration
- [ ] Multi-model comparison reports
- [ ] Continuous compliance monitoring (drift alerts)
